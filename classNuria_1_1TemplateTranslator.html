Detailed Description

Translator interface for TemplateEngine.

The Twig module implements a extension to the Twig language which allows to have easy access to translations from the Twig code.

Accessing variables

Using string interpolation, it's possible to access variables (and execute Twig functions) from translations, for example:

Hello #{user.name|capitalize}
